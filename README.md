# Calmavet Web

Sitio estático de Calmavet para servicios de eutanasia e incineración de mascotas a domicilio en Barcelona y Cataluña.

## Estructura

- `index.html`: landing principal.
- `barcelona/index.html`: página SEO local para Barcelona.
- `css/styles.css`: estilos compartidos mobile-first.
- `images/`: recursos visuales disponibles.

## Notas

- El diseño utiliza las imágenes incluidas en `images/`.
- Los testimonios de la primera versión están marcados como representativos y deben sustituirse por reseñas verificadas antes de publicar.
